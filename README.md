# Spinning Drunken Pirates

A short 2D arcade game created during a 7-day Game Jam. Test your accuracy, manage risk, and try to hit the target while your crosshair becomes increasingly unpredictable with every level of intoxication.

## 🎯 Game Objective & Gameplay

The goal is simple: survive as many rounds as possible and reach the required score before you run out of throws. The catch is that with each round, the score threshold increases, and the game becomes progressively harder.

The gameplay is divided into two phases:
1. **Rolling Phase (Wheel of Fortune):** At the start of each round, you roll for your stats: 
   * Number of throws (daggers),
   * Point multiplier,
   * Drunk Level – which increases and stacks with each round.
2. **Throwing Phase (Target):** Using your crosshair, you throw daggers at a spinning target. The rolled Drunk Level drastically affects the crosshair's behavior – the higher it is, the more it sways, and the more aggressively the target spins.

If you reach the required point threshold by the end of the round, you progress and roll again. If not – Game Over.

## 🕹️ Controls
* **Mouse:** Move the crosshair around the screen.
* **Spacebar (Hold):** Focus and charge shot precision (the crosshair pulses and changes size).
* **Spacebar (Release):** Execute the throw.
